# Kitty

## Kitty
Instalamos kitty
```bash
sudo pacman -S kitty
```
Con el comando `kitten theme` elegimos el tema.

La con esto se nos generan 3 ficheros en el directorio de configuración
```bash
theasker@theasker-pc ~ $ tree .config/kitty/
.config/kitty/
├── Aquarium Dark.conf
├── current-theme.conf
└── kitty.conf
```


Además, se puede abrir la config con `Ctrl+Shift+F2` dentro de kitty
